Resumen General de la Versión

Este release realiza un ajuste preventivo en la automatización de la plataforma PediGrido, pausando la sincronización automática de sucursales para asegurar la estabilidad operativa durante periodos de prueba. Adicionalmente, introduce formalmente el sistema de versionado en la API para una mejor trazabilidad.

Análisis Técnico Detallado

api/src/controllers/branch.js

Suspensión de Tareas Automáticas PediGrido

Se han suspendido temporalmente los crons encargados de la sincronización diaria de información de sucursales y el chequeo de horarios forzados. El código permanece en el sistema (comentado) bajo el tag [OpenClosePG] para facilitar su reactivación futura.

Crons PediGrido:
// ANTES (Activos)
cron.schedule(scheduleGetInfoBranchesGrido, async () => await getInfoBranchesGrido());
cron.schedule(scheduleforcedCronHorarios, async () => await checkforcedCronHorarios());

// DESPUÉS (Desactivados temporalmente)
// [OpenClosePG] cron.schedule(scheduleGetInfoBranchesGrido, async () => await getInfoBranchesGrido());
// [OpenClosePG] cron.schedule(scheduleforcedCronHorarios, async () => await checkforcedCronHorarios());

Establecimiento de Control de Versiones
