Resumen General de la Versión

Este release activa formalmente la automatización de PediGrido, permitiendo el monitoreo automático de locales y la sincronización de horarios. También mejora la robustez de los procesos de sincronización para evitar ejecuciones redundantes.

Activación de Automatización y Sincronización Robustez

Se activaron las tareas programadas (crons) para la integración con PediGrido (OpenClosePG). Además, se reordenó la lógica de actualización de flags de sincronización forzada para prevenir concurrencias innecesarias en procesos largos.

Crons PediGrido:
// ANTES
//[OpenClosePG] cron.schedule(schedulePediGridoOffline, async () => await checkOfflinePOSpedigrido());
//[OpenClosePG] cron.schedule(scheduleGetInfoBranchesGrido, async () => await getInfoBranchesGrido());

// DESPUÉS (Activación formal)
cron.schedule(schedulePediGridoOffline, async () => await checkOfflinePOSpedigrido());
cron.schedule(scheduleGetInfoBranchesGrido, async () => await getInfoBranchesGrido());
Gestión de Flags:
// DESPUÉS (Optimización: Reseteo de flag ANTES de la ejecución)
async function checkforcedCronHorarios() {
  let forcedCronHorarios = await configsModel.findOne({});
  if (!forcedCronHorarios.forcedCronHorarios) return;
  
  await configsModel.updateOne({}, { forcedCronHorarios: false });
  await getInfoBranchesGrido();
}
